# 我的第一篇技術文章

歡迎來到我的技術部落格！這是我使用 Astro 寫的第一篇文章。

## 為什麼選擇 Astro？

Astro 是一個現代化的靜態網站生成器，它有以下優點：

### 🚀 零 JavaScript 預設
- 預設情況下不載入任何 JavaScript
- 更快的載入速度和更好的 SEO
- 只在需要時才載入互動功能

### 📝 內容集合
- 類型安全的內容管理
- 自動生成 TypeScript 類型
- 支援多種內容格式（Markdown、MDX）

### 🎨 靈活的框架整合
- 可以與 React、Vue、Svelte 等框架整合
- 支援多種 UI 框架
- 豐富的整合生態系統
